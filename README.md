# Markdown Editor

A simple, elegant markdown editor with real-time preview functionality built with vanilla JavaScript.

## Features

- **Real-time Preview**: See your markdown rendered as you type
- **Split-pane Interface**: Write markdown on the left, see preview on the right
- **Formatting Toolbar**: Quick buttons for common markdown syntax
  - Bold
  - Italic
  - Headings
  - Links
  - Code
- **Auto-save**: Content is automatically saved to browser's local storage
- **Responsive Design**: Works on desktop and mobile devices
- **Clean UI**: Modern, distraction-free interface

## Usage

1. Open `index.html` in your web browser
2. Start typing markdown in the left pane
3. See the formatted preview in the right pane
4. Use toolbar buttons for quick formatting
5. Your content is automatically saved and will persist between sessions

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [marked.js](https://marked.js.org/) - Fast markdown parser

## Supported Markdown Syntax

- Headers (H1-H6)
- Bold and italic text
- Lists (ordered and unordered)
- Links
- Images
- Code blocks and inline code
- Blockquotes
- Tables
- Horizontal rules

## Getting Started

Simply open `index.html` in your browser - no build process or dependencies required!
